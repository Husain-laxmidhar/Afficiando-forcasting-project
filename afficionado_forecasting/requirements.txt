# Core requirements for the Streamlit app (app.py) and the training pipeline.
# Deliberately excludes prophet / pmdarima / xgboost: they need compiled
# build toolchains (cmdstan, Fortran) that commonly fail on Streamlit
# Community Cloud's build image. A failed build aborts the ENTIRE
# `pip install -r requirements.txt` step, which is why even unrelated
# packages like plotly can end up "not installed" -> ModuleNotFoundError.
# app.py only uses the models in SERIES_MODELS (Naive/Seasonal Naive/
# Moving Average/SARIMA/Exponential Smoothing), none of which need these.
# If you want Prophet/XGBoost for local experimentation, install them
# separately (see requirements-extra.txt) -- do not add them back to this
# file if you deploy on Streamlit Community Cloud.
pandas>=2.2,<2.3
numpy>=1.26,<1.27
pyarrow>=15.0,<16.0
statsmodels>=0.14,<0.15
plotly>=5.20,<5.21
streamlit==1.35.0
scikit-learn>=1.4,<1.5
matplotlib>=3.8,<3.9
